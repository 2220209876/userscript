BaiduPanDownloadHelper
======================

批量导出百度盘的下载链接

![效果图](百度盘脚本.png)

## 使用说明

支持Firefox（Greasemonkey/Scriptish）， Chrome（Tampermonkey）

非常感谢 [BaiduPanMD5Button](https://userscripts.org/scripts/show/156906)
